## Usage

### Install Development Requirements
```bash
pip install -r requirements-dev.txt
```

### Run Linting
```bash
# Check for style issues
flake8 video_processor.py

# Fix formatting
black video_processor.py

# Sort imports
isort video_processor.py

# Detailed linting
pylint video_processor.py
```

### Type Checking
```bash
mypy video_processor.py --ignore-missing-imports
```

### Run Tests
```bash
pytest
```

### Generate Coverage Report
```bash
pytest --cov=. --cov-report=html
```

## Pre-commit Setup (Optional)

To automatically run checks before committing:

1. Install pre-commit: `pip install pre-commit`
2. Create `.pre-commit-config.yaml` in the project root
3. Run: `pre-commit install`
4. On next commit, checks will run automatically
